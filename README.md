# Smart Renewable Energy Report Analyzer

## Project Overview

The Smart Renewable Energy Report Analyzer is an NLP-based web application developed using Python and Streamlit. The project analyzes renewable energy reports and provides intelligent insights such as sentiment analysis, report summarization, keyword visualization, and chatbot interaction.

This project was developed as part of an internship milestone project to demonstrate skills in Python, NLP, Streamlit, and data visualization.

---

## Features

- Upload TXT and PDF renewable energy reports
- Sentiment Analysis (Positive / Negative / Neutral)
- Automatic Report Summarization
- Keyword Word Cloud Visualization
- Download Analyzed Report
- Interactive Energy Report Chatbot
- User-Friendly Streamlit Dashboard

---

## Technologies Used

- Python
- Streamlit
- TextBlob
- Sumy
- WordCloud
- Matplotlib
- PyPDF2
- NLP (Natural Language Processing)

---

## Project Workflow

1. User uploads renewable energy report
2. System extracts report content
3. NLP processes report text
4. Sentiment analysis is performed
5. Important summary is generated
6. Word cloud visualizes important keywords
7. Chatbot answers user queries
8. Final analyzed report can be downloaded
